# AZL Consulting and Trading

## Overview
A modern, multi-page professional website for an accounting and advisory firm. Built with React and Tailwind CSS over HTML.

## Pages & Structure
- **Home**: Main landing page highlighting primary value propositions.
- **About Us**: Information about the firm's approach and core values.
- **Contact Us**: Address, hours, and contact inquiry form.
- **Services Pages**:
  - `accounting-compliance.html`
  - `audit-assurance.html`
  - `business-advisory.html`
  - `corporate-finance-valuations.html`
  - `tax-private-business.html`

## Technical Details
- Framework: React 18 (Standalone)
- Styling: Tailwind CSS (Standalone)
- Icons: Lucide (Static CSS)
